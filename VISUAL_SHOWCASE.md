## 🎯 INTERACTION FLOWS

### Form Flow
```
1. User arrives at form
2. Step 1: Select time
3. Next button enabled
4. Step 2: Pick interests
5. Next button enabled if interests selected
6. Step 3: Choose walking level
7. Next button enabled
8. Step 4: Select crowd preference
9. Create Tour button enabled
10. Form submits, loading spinner shows
11. Tour displays
```

### Tour Navigation Flow
```
1. User views tour overview
2. Clicks Stops tab
3. See all stops in grid
4. Click a stop to navigate
5. Jump to that stop
6. Click Adjust tab
7. Choose adjustment
8. Loading spinner
9. Success message
10. Back to form button available
```
